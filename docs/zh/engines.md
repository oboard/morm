# 引擎总览

引擎层负责把查询语义转换为真实数据库行为，包括 SQL 渲染、参数绑定、执行与结果返回。

## 内置引擎

- SQLite
- MySQL
- PostgreSQL
- SQL Server
- Oracle
- MongoDB
